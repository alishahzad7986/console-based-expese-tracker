

# 💰 Console-Based Expense Tracker

A simple and efficient **console-based expense tracker** built to help users manage daily expenses directly from the terminal. This project demonstrates core programming concepts like file handling, data structures, and user input validation in a practical way.

---

## 🚀 Features

* ➕ Add new expenses with details (amount, category, description)
* 📋 View all recorded expenses
* 🔍 Search expenses by name or category
* ❌ Delete expenses بسهولة
* 💾 Data persistence using file storage (JSON/CSV)
* 📊 Simple and user-friendly console interface

Expense tracker applications like this typically allow users to record, organize, and analyze spending data efficiently in a lightweight environment ([GitHub][1])

---

## 🛠️ Tech Stack

* Python (Core Logic)
* File Handling (JSON / CSV)
* CLI (Command Line Interface)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/alishahzad7986/console-based-expese-tracker.git
```

2. Navigate to the project folder:

```bash
cd console-based-expese-tracker
```

3. Run the program:

```bash
python main.py
```
